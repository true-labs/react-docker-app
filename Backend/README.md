# docker-intro-backend
Sample API in golang for our docker intro session

### Installation

Install Dependencies 
``` bash
$ make dep
```


### Run Application

Run The api
``` bash
$ make run
```


### Test Application

test your applicaiton
``` bash
$ make test
```

###
check Docker branch for details on how to run with docker image
